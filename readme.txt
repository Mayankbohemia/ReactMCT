
# Movie App
  This react movie app renders the top 100 movies and some awesome filtering methods. 
  This application can also render movie trailers.

# Features 
  - Contains a Home Page where top 100 movies are rendered.
  - Contains a search input to see movie of our choice
  - Contains genre buttons to filter the movies of a particular genre 
  - Contains a route link through which we can see the trailer of a movie
